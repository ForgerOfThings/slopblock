This is a very basic firefox extension i am building to block AI content on youtube, to discuss the development, or otherwise support/communicate with me, please join https://www.discord.gg/kpBYYA5Nuw

To switch from delete mode to label mode, go to manage extension > preferences > filter option and select label from the drop down and then save

What does it block?

- AI generated video summaries
- Channels we have logged as AI
